The Game Loop

How It Works

Every real UFC event generates new versioned fighter NFTs. Collect, stake, trade, burn, and compete in a self-reinforcing economy.

01
Collect

UFC event announced. New fighter versions drop as Standard, Limited, and First Edition NFTs. Each appearance creates a new version.

02
Compete

Enter the Arena. Lock your fighters in prediction pools tied to real bouts. LGNDs compete for prestige slots with steal mechanics.

03
Burn

Collect every version of a fighter, then burn them all to forge a Legendary. The scarcest version caps how many LGNDs can ever exist.

04
Dominate

LGNDs unlock the highest Arena tiers, the biggest reward pools, and permanent prestige. Supply only goes down. Value only goes up.

Two-Token Economy

$FIGHT + $ARENA

Two tokens, two purposes. $FIGHT fuels every transaction. $ARENA governs the ecosystem. Together they create a self-reinforcing flywheel.

$FIGHT
Utility / Fuel

The transactional engine. Every player action consumes $FIGHT. Fees are split between permanent burns (deflationary) and recycled rewards (back to stakers).

  • 10-15% discount on SE purchases
  • Required for LE Launchpad
  • LGND upgrade fee (100% burn)
  • Arena entry & predictions
  • Escrow trade fees
  • Burn + recycle on every transaction
$ARENA
Governance / Ownership

Ownership of the Fight Farm ecosystem. $ARENA holders control treasury decisions, reward distribution, and the future direction of the platform.

  • Required for FE auctions (1-of-1s)
  • Treasury governance votes
  • Reward distribution rules
  • Ecosystem upgrade proposals
  • Revenue share (future)

Build Plan

Roadmap

Four phases. From genesis mint to a fully autonomous fight economy.

Phase 1 — Foundation
Genesis Drop
  • $FIGHT and $ARENA token launch on Base
  • Fighter NFT contract (ERC-721) deployment
  • SE minting for first UFC event
  • Fighter Registry + metadata on IPFS
  • Basic staking (EmissionsVault)
Phase 2 — The Arena
Predictions Go Live
  • Arena fight pools + on-chain settlement
  • LE Launchpad (random, $FIGHT only)
  • FE 1-of-1 auctions ($ARENA only)
  • FeeRouter + burn/recycle engine
  • Escrow trade board (NFT-for-NFT)
Phase 3 — Burn Era
Legendaries Unlocked
  • LGND upgrade system (burn all versions)
  • LGND Arena slots + steal mechanics
  • Bottleneck analytics dashboard
  • AI Matchmaker agent (automated event detection)
  • DAO governance via Snapshot + Safe
Phase 4 — Domination
Full Autonomy
  • Custom marketplace UI
  • Full agent automation (FighterFactory, NFTBuilder)
  • Cross-event meta and fighter career tracking
  • Revenue share for $ARENA holders
  • Expansion beyond UFC
